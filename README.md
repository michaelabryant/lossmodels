# lossmodels

A Python library for actuarial loss modeling, including frequency, severity, and aggregate risk models.

## Overview

`lossmodels` provides tools for modeling insurance losses using classical actuarial methods:

- Frequency distributions (claim counts)
- Severity distributions (claim sizes)
- Aggregate loss models (collective risk models)
- Coverage modifications (deductibles, limits)
- Risk measures (VaR, TVaR, stop-loss)

The library is inspired by *Loss Models: From Data to Decisions* (Klugman, Panjer, Willmot) and is designed to be:

- simple and transparent
- mathematically grounded
- easy to extend
