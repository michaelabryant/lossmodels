# lossmodels Documentation

## Overview

`lossmodels` is a Python library for actuarial loss modeling using frequency-severity methods.

This documentation covers:

- Frequency models
- Severity models
- Coverage modifications
- Aggregate modeling
- Estimation and model selection
- Credibility theory
- Empirical distributions
- Risk measures
- Worked examples
